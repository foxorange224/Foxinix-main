¿Qué es Arch Linux?

Arch Linux es una distribución Linux independiente que ofrece lo último en software con un modelo rolling release. Ahora, con archinstall, la instalación es más fácil que nunca.

  • Rolling Release: Siempre tienes la última versión
  • archinstall: Instalador oficial moderno y gráfico
  • Minimalista: Instalas solo lo que necesitas
  • Bien documentada: La Wiki de Arch es excelente

Requisitos del Sistema

  • Procesador: 64-bit (x86_64)
  • RAM: 512 MB (2 GB recomendados)
  • Espacio: 10 GB (20 GB recomendados)
  • Conexión: Ethernet o WiFi

Preparación del USB Bootable

  1. Descarga la ISO desde archlinux.org
  2. Usa Rufus (Windows) o Etcher (Linux/Mac)
  3. Selecciona la ISO y el USB, luego clic en "Start"
  4. Arranca desde USB en tu PC (F12/F2 para menu boot)

Conectar WiFi con iwctl

Si usas WiFi (no Ethernet), usa iwctl para conectar:

# Entrar al modo interactivo
iwctl

# Ver adaptadores disponibles
[iwd#agent] device list

# Escanear redes
[iwd#agent] station wlan0 scan

# Ver redes encontradas
[iwd#agent] station wlan0 get-networks

# Conectar a tu red (reemplaza 'MiRed' y 'password')
[iwd#agent] station wlan0 connect "MiRed"

# Salir
exit

# Verificar conexión
ping -c 3 archlinux.org

Nota: Si usas Ethernet, te conectarás automáticamente.

Particionar el Disco (Opcional)

Si quieres particionar manualmente (archinstall puede hacerlo automáticamente):

# Ver discos
lsblk

# Particionar con cfdisk
cfdisk /dev/sda

# Crear particiones:
# - /dev/sda1: Boot (512MB, bootable)
# - /dev/sda2: Swap (2GB)
# - /dev/sda3: Root (resto, tipo Linux)

# Formatear
mkfs.ext4 /dev/sda3
mkfs.ext4 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2

# Montar
mount /dev/sda3 /mnt
mount --mkdir /dev/sda1 /mnt/boot

Instalación con archinstall

¡Esta es la forma moderna y recomendada!

# Montar particiones (si no lo hiciste)
mount /dev/sda3 /mnt
mount --mkdir /dev/sda1 /mnt/boot

# Ejecutar el instalador interactivo
archinstall

Configuración en el instalador:

  1. Idioma: Spanish (es)
  2. Distribución: Arch Linux
  3. Mirror region: Selecciona tu país
  4. Disk layout: "Best" o selecciona manualmente
  5. Filesystem: ext4 o btrfs
  6. Bootloader: systemd-boot o GRUB
  7. Swap: hibernation o zram
  8. Hostname: MiArch
  9. Root password: Establece una contraseña
  10. User account: Crea un usuario
  11. Profile: Desktop o Minimal
  12. Audio: pipewire o pulseaudio
  13. Network: NetworkManager
  14. Timezone: America/Caracas (o tu zona)
  15. Install: Confirmar instalación

¡Listo! El instalador hará todo el trabajo automáticamente.

Instalar Entorno de Escritorio

Si elegiste "Minimal" en archinstall,instala tu escritorio:

Gnome (completo):

sudo pacman -Syu
sudo pacman -S gnome gnome-extra
sudo systemctl enable gdm
sudo systemctl start gdm

KDE Plasma (moderno):

sudo pacman -Syu
sudo pacman -S plasma kde-applications
sudo pacman -S sddm
sudo systemctl enable sddm
sudo systemctl start sddm

XFCE (ligero):

sudo pacman -Syu
sudo pacman -S xfce4 xfce4-goodies
sudo pacman -S lightdm lightdm-gtk-greeter
sudo systemctl enable lightdm
sudo systemctl start lightdm

Drivers:

# NVIDIA
sudo pacman -S nvidia nvidia-utils nvidia-settings

# AMD
sudo pacman -S xf86-video-amdgpu

# Intel
sudo pacman -S xf86-video-intel

Configuración Final

Comandos esenciales:

  • sudo pacman -Syu - Actualizar sistema
  • sudo pacman -S nombre - Instalar programa
  • sudo pacman -R nombre - Eliminar programa
  • paru -S nombre - Instalar desde AUR

Instalar AUR (paru):

sudo pacman -S git base-devel
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si
sudo pacman -Syu && paru -Syu

WiFi después de instalar:

# Usando NetworkManager (ya instalado)
sudo systemctl enable NetworkManager
sudo systemctl start NetworkManager
nmcli device wifi connect "MiRed" password "mipassword"
